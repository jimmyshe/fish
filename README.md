# Fish Pet 🐟

一个基于 Electron + Vite + Vue 3 制作的桌面摸鱼宠物，陪伴你度过漫长的上班时光，并在下班时提醒你"快跑"。

## ✨ 功能特性

### 🕒 下班倒计时
实时计算距离下班还有多少时间，精确到分钟。

### 🐟 动态心情系统
小鱼会根据距离下班的时间自动切换心情、颜色和动画：

| 心情 | 触发条件 | 颜色 | 状态 |
|------|----------|------|------|
| **平静** | 2小时以上 | 蓝色 | 安心摸鱼 |
| **开心** | 不到 2 小时 | 黄色 | 心情愉悦 |
| **紧张** | 不到 30 分钟 | 粉色 | 流汗冲刺 |
| **悲伤** | 加班中 | 紫色 | 流泪哭泣 |

### 🖱️ 鼠标互动

- **眼睛追踪**：小鱼的眼睛会实时跟随鼠标移动
- **悬停特效**：鼠标悬停在鱼身上时，小鱼会加速游动并发出金色光晕
- **点击互动**：点击小鱼（非拖拽）会触发随机趣味消息并飘出爱心 / 星星粒子

### 😱 受惊模式
鼠标移动过快时，小鱼会：
- 瞳孔变成竖形杏仁眼
- 身体变红并剧烈抖动
- 喊出"不要过来！"等惊叫
- 飘出惊吓粒子

### 😴 睡眠模式
**3 分钟**无鼠标移动后，小鱼自动入睡：
- 眼睛变成闭合的弧线
- 身体变暗，漂浮变缓慢
- 气泡显示 `Z z z ...`，并持续飘出 Zzz 粒子
- 任何鼠标移动可唤醒

### 🌈 连击彩蛋
**3 秒内点击 5 次**解锁隐藏彩蛋：
- 小鱼头顶出现皇冠
- 全身颜色持续彩虹旋转（360° hue-rotate）
- 飘出 🌈✨🎉 粒子
- 连击数从 2 次起显示 `Nx COMBO!` 徽章

### 💬 随机自言自语
小鱼每隔 1.5~3.5 分钟会随机冒出一句话，例如：
> "鱼的记忆只有7秒... 等等刚才说啥来着"
> "老板是什么味道的..."
> "静静地看着你们上班"

### 🏊 自主漂移
小鱼每隔 35~70 秒会自动游向屏幕上的新位置，带有平滑过渡动画。

### 其他
- **自由拖拽**：左键按住可拖动到屏幕任意位置
- **右键菜单**：设置下班时间 / 显隐窗口 / 退出
- **系统托盘**：最小化后可从托盘图标操作
- **持久化设置**：下班时间保存到本地，重启后恢复

## 🛠️ 技术栈

- [Electron](https://www.electronjs.org/) - 跨平台桌面应用框架
- [electron-vite](https://electron-vite.org/) - Electron 专用构建工具
- [Vue 3](https://vuejs.org/) - 渐进式 JavaScript 框架
- TypeScript
- Node.js `fs` + `app.getPath('userData')` 持久化配置（无第三方存储依赖）

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 运行开发环境

```bash
npm run dev
```

### 构建与打包

仅构建代码：
```bash
npm run build
```

构建并打包为可执行文件（Windows `.exe` / macOS `.dmg`）：
```bash
npm run dist
```

## 🎮 使用说明

1. 启动后，一只半透明的小鱼会出现在桌面右下角。
2. **左键拖拽**：按住小鱼可随意移动位置。
3. **左键点击**：触发互动消息和粒子特效；快速点击 5 次解锁彩蛋。
4. **右键菜单**：设置下班时间、显隐窗口或退出应用。
5. **别动鼠标**：静置 3 分钟，小鱼会自己睡着。
6. **快速挥动鼠标**：吓一吓小鱼看看效果。
