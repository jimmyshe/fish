# Fish Pet 🐟

一个基于 Electron + Vite + Vue 3 制作的桌面摸鱼宠物，陪伴你度过漫长的上班时光，并在下班时提醒你“快跑”。

## ✨ 功能特性

- 🕒 **下班倒计时**：实时计算距离下班还有多少时间。
- 🐟 **动态小鱼**：小鱼会根据距离下班的时间长短改变心情、颜色和游动动画：
    - **平静 (Normal)**：时间还早，安心摸鱼。
    - **开心 (Happy)**：距离下班不到 2 小时，心情愉悦。
    - **紧张 (Nervous)**：距离下班不到 30 分钟，激动流汗准备冲刺。
    - **悲伤 (Sad)**：加班时间，小鱼会流下伤心的泪水。
- 💬 **对话互动**：小鱼会展示对话气泡，提示剩余的上班时间，并根据你的状态给出有趣的提醒。
- 🖱️ **自由拖拽**：支持左键按住小鱼并将其拖动到屏幕的任意位置。
- ⚙️ **自定义时间**：支持自定义设置每天的下班时间。

## 🛠️ 技术栈

- [Electron](https://www.electronjs.org/) - 跨平台桌面应用开发框架
- [Vite](https://vitejs.dev/) - 下一代前端构建工具
- [Vue 3](https://vuejs.org/) - 渐进式 JavaScript 框架
- TypeScript

## 🚀 快速开始

### 1. 克隆或下载项目

### 2. 安装依赖

```bash
npm install
```

### 3. 运行开发环境

```bash
npm run dev
```

### 4. 构建与打包

仅构建代码：
```bash
npm run build
```

构建并打包为可执行文件（如 Windows 的 `.exe` 或 macOS 的 `.dmg`）：
```bash
npm run dist
```

## 🎮 使用说明

1. 启动应用后，一只半透明的小鱼会出现在你的桌面上。
2. **左键拖拽**：按住小鱼可以随意移动它的位置。
3. **右键菜单**：右键点击小鱼可以打开菜单，选择设置下班时间或退出应用。
4. **设置时间**：在设置面板中输入你的下班时间并保存，小鱼的状态将实时更新。
